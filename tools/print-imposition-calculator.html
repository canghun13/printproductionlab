Print Imposition Calculator

Compare two orientations using usable sheet geometry. Values use one consistent unit (mm, in, etc.).

Enter dimensions, then calculate.

Method

Usable width is parent width minus two outer margins; usable height is parent height minus margin and gripper. Each finished item occupies its trim size plus bleed on both edges. The calculator floors across and down counts for normal and rotated layouts.

Worked example

A 700 × 500 sheet, 90 × 50 finished item, 3 bleed, 50 outer margin and 12 gripper is compared in both orientations.

Preview is proportional and informational, not production-certified imposition. Confirm gutters, marks, grain direction, press gripper and finishing requirements.
